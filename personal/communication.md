# Communication Style

**Author:** Anderson Henrique da Silva

---

## Preferences

### Tone
- Professional but not formal
- Direct without being terse
- Technical precision valued

### Depth
- **Learning Mode:** Full explanations, context, alternatives
- **Execution Mode:** Minimal, focused, actionable
- **Review Mode:** Critical analysis, edge cases

### Language
- **Portuguese:** Ideation, brainstorming, informal discussion
- **English:** Code, documentation, technical writing, commits

### Format
- Markdown for documentation
- Code blocks with language identifiers
- Tables for comparisons
- Diagrams when structure matters
